# Módulo de Contagem (Count)

Este diretório marca a **Contagem** como a primeira engine operacional vinculada à visão do MOC (Módulo Operacional do Colaborador).

**Aviso Importante:**
- Na etapa atual, este diretório atua apenas como um marcador organizacional.
- **Não houve migração lógica** da engine de contagem nesta fase; o código e o comportamento do sistema permanecem inalterados.
- Este espaço está reservado para a futura centralização dos serviços e componentes de inventário.
