# MOC Core

Este diretório contém o núcleo estrutural do **Módulo Operacional do Colaborador**.

O MOC foi projetado para atuar como um contâiner organizacional para rotinas operacionais (Contagem, Checklist, etc.). 

**Status da Migração:**
- A contagem é a primeira rotina mapeada para este ecossistema.
- Nesta fase inicial, a engine de contagem e seus fluxos atuais permanecem intactos em seus diretórios originais, sem migração lógica realizada.
- Este diretório serve apenas como base estrutural para a evolução futura.
